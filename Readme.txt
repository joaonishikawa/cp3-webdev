Inscrição do Projeto:
Este projeto consiste na criação de um catálogo online que permite ao usuário interagir com uma lista de produtos. O catálogo é construído utilizando manipulação de DOM, arrays e objetos, permitindo a filtragem e organização dos produtos por categoria e disponibilidade.
Regras Gerais

Tecnologias Utilizadas
HTML: Estruturação da interface do usuário.
CSS: Estilização dos elementos da interface.
JavaScript: Lógica de manipulação de DOM, arrays e objetos.

Funcionalidades

Array de Produtos:
 Um array de objetos que contém pelo menos 10 produtos, cada um com as seguintes propriedades:
    nome
    preco
    categoria
    disponibilidade (true/false)

    Interface do Usuário:
    Botão para listar todos os produtos.
    Formulário com um select para categorias e um checkbox "Somente disponíveis".
    Botão "Filtrar" para aplicar os filtros selecionados.

    Interatividade:
    Ao clicar no botão "Filtrar", os produtos são listados de acordo com a categoria selecionada e a disponibilidade.
    Destaque visual dos produtos ao passar o mouse sobre eles.

    Funcionalidades adicionais:
    Agrupamento dos produtos por categoria com títulos de seção.
    Opção de mostrar produtos disponíveis ou não disponíveis no filtro.
    Ordenação dos produtos por preço.
